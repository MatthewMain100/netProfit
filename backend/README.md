# Backend

Каталог содержит серверные и расчетные артефакты проекта.

## Содержимое

- `db.js` - подключение к PostgreSQL.
- `migrate.js` - применение миграций.
- `migrate_status.js` - проверка состояния миграций.
- `rebuild_projections.js` - восстановление проекций из журнала событий.
- `seed_demo_events.js` - наполнение демонстрационного контура.
- `db/migrations/` - миграции схемы.
- `db/schema/` - базовая SQL-схема.
- `db/seeds/` - стартовые наборы данных.

Скрипты ориентированы на рабочий каталог `backend/`, поэтому структура `db/*` закреплена на уровне репозитория.
