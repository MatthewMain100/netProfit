# Frontend

Каталог выделен под клиентский контур системы.

В составе публичного репозитория сохранены:

- материалы по структуре интерфейса;
- зафиксированные экраны модулей в `../docs/screenshots/`;
- привязка клиентского контура к общему регламенту ветвления и релизов.

Функциональные изменения клиентской части должны вестись в ветках `feature/*` с последующей интеграцией через `develop`.
