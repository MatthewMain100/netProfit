# Frontend

Клиентский контур проекта на React и Vite.

## Состав

- `src/pages/` - прикладные страницы модулей.
- `src/components/` - переиспользуемые UI-компоненты.
- `src/api.js` - обвязка для вызова backend API.
- `src/App.jsx` - маршрутизация и основной layout.
- `public/` - статические файлы.

## Запуск

```powershell
npm install
npm run dev
```

Frontend ожидает backend API на `http://localhost:4000`.

Скриншоты актуальных экранов вынесены в [docs/screenshots](../docs/screenshots).
